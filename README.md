

# Sistema de Loja de Eletrodomésticos

Projeto simples em Python para simular a compra de produtos em uma loja de
eletrodomésticos, utilizando diferentes formas de pagamento no terminal.

## Funcionalidades
- Exibição dos produtos disponíveis
- Menu interativo no terminal
- Pagamento à vista com desconto
- Pagamento parcelado com e sem juros
- Histórico de compras

## Tecnologias utilizadas
- Python
  
## Porcentagem de uso da IA
- 5%
- Uso foi responsável pelo calculo de juros e parcela
- Função que foi usada pagar_parcelado_12_vezes()
- Codigo gerado da linha 73 a 78
